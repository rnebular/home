# Mealie
I dislike Pinterest based recipes because they are always loaded with ads, click bait and annoying pop-up videos. I want to use Mealie to capture my favorite recipes so I can use them in a clean format, as well as better sort and manage them. The added bonus is that sometimes recipe sites will remove a recipe after a certain amount of time, breaking the pin. This is a nice backup for those recipes.
