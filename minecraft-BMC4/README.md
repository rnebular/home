# minecraft-ec2
Configuration of a Minecraft Server on Ubuntu 24.04.4 LTS, running as a Docker container.

Place docker-compose.yml into folder where minecraft will run. This file is setup to use Auto Forge (CourseForge) and the Better Minecraft MODPACK, BMC4, version `1.20.1`.

Ref: [BMC4-Modpack](https://www.curseforge.com/minecraft/modpacks/better-mc-forge-bmc4)

Uses Docker image: [itzg/docker-minecraft-server](https://github.com/itzg/docker-minecraft-server)

Get CourseForge AUTOFORGE API KEY here:
https://console.curseforge.com/?#/api-keys
